## 1. 프로젝트 한 줄 요약

비영리 민간단체 **교육성장네트워크 꿈들**의 공식 홈페이지를,
기존 회원형 PHP 사이트(`꿈들.com`)에서 **가볍고 무료로 운영 가능한 정적 사이트**로
새로 만들고 콘텐츠를 이관하는 프로젝트.

- **목표:** 소개·콘텐츠 위주의 정적 홈페이지 + GitHub Pages 무료 배포
- **지금 단계:** 기본 사이트 골격 + Claude 스타일 디자인까지 완성됨.
  다음 할 일은 **디자인 고도화**와 **게시글 본문 이관**.

## 2. 기술 스택 (중요 — 이 범위를 지켜주세요)

- **순수 정적 사이트**: HTML + CSS + 바닐라 JavaScript
- **빌드 도구 없음** (npm/webpack/React 등 사용하지 않음). 파일을 그대로 브라우저에서 열면 동작.
- **프레임워크·백엔드 없음**: 서버, DB, 로그인 기능 없음 (정적 호스팅 전제)
- **배포 대상**: GitHub Pages
- **언어**: 한국어 (`<html lang="ko">`)
- **반응형**: 모바일 우선(Mobile First)

> ⚠️ Codex에게: React/Next.js/빌드 파이프라인으로 **바꾸지 마세요.**
> 반드시 **빌드 불필요한 정적 HTML/CSS/JS** 구조를 유지해야 GitHub Pages에서
> 그대로 무료로 굴러갑니다.

## 3. 저장소 구조

```
/
├─ index.html          # 홈 (히어로 · 통계 · 주요활동 · 최근소식 · 후원CTA)
├─ about.html          # 단체소개 (미션 · 연혁)
├─ projects.html       # 사업·프로젝트 (하늘오름어린이풍물단 등)
├─ news.html           # 소식·보도자료·공지 목록 (카테고리 필터)
├─ sponsor.html        # 후원안내
├─ contact.html        # 문의
├─ 404.html            # 404 페이지
├─ .nojekyll           # GitHub Pages가 Jekyll 처리 안 하도록
├─ assets/
│  ├─ style.css        # 공통 디자인(전부 여기서 조절) — CSS 변수 기반
│  ├─ site.js          # 공통 헤더/푸터 자동 주입 + 모바일 메뉴
│  └─ posts.js         # 게시글 데이터(배열) + 렌더링 함수
├─ README.md           # 사용자용 안내(배포/이관 방법)
└─ CODEX.md            # (이 문서)
```

### 핵심 파일 설명

- **`assets/style.css`** — 디자인의 90%가 여기서 결정됩니다.
  최상단 `:root`에 색상/폰트/여백 CSS 변수가 정의되어 있어 색·톤을 한 번에 바꿀 수 있음.
  현재 테마는 **Claude(Anthropic) 스타일**: 따뜻한 크림(ivory) 배경 + 코랄(clay) 포인트.
- **`assets/site.js`** — 모든 페이지의 `<div id="site-header"></div>` /
  `<div id="site-footer"></div>` 자리에 헤더·푸터를 JS로 주입합니다.
  → **메뉴/푸터를 바꾸려면 이 파일 하나만** 수정하면 전 페이지에 반영됨.
  네비게이션 항목은 `NAV` 배열에 정의되어 있음.
- **`assets/posts.js`** — 게시글을 데이터 배열 `POSTS`로 관리.
  항목을 추가하면 홈 '최근 소식'과 `news.html`에 자동 표시됨. (아래 6번 참고)

## 4. 로컬 미리보기 & 배포

### 미리보기 (로컬)
정적 파일이라 아무 로컬 서버로 열면 됩니다. 예:
```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
> `file://`로 바로 열어도 대부분 동작하지만, `site.js`/`posts.js`가 JS로
> 콘텐츠를 주입하므로 로컬 서버로 여는 것을 권장.

### 배포 (GitHub Pages)
1. 작업 브랜치를 `main`에 머지
2. GitHub 저장소 → **Settings → Pages**
3. **Source: Deploy from a branch** → `main` / `/(root)` → Save
4. `https://<사용자명>.github.io/kkumddel/` 에 공개
5. (선택) Custom domain에 `꿈들.com` 연결 + DNS 설정

## 6. 게시글 데이터 구조 (`assets/posts.js`)

`POSTS` 배열에 객체를 추가하면 됩니다:
```js
{
  category: "소식",          // 소식 | 보도자료 | 프로젝트 | 공지사항 중 하나
  title: "글 제목",
  date: "2026-06-30",        // YYYY-MM-DD (최신순 자동 정렬)
  body: "본문 내용...",       // 기존 사이트 본문 (선택)
  link: "https://..."        // 외부 원문 링크 있으면 (선택, 있으면 새 탭으로 열림)
}
```
- 렌더링 함수 `renderPosts(targetId, {category, limit})` 가 목록을 그림.
- 카테고리 값은 CSS 태그 색상(`.t-소식`, `.t-보도자료` 등)과 연결되어 있으니
  **새 카테고리를 추가하면 `style.css`에도 `.post .tag.t-<이름>` 색을 추가**할 것.
- 현재는 목록만 표시되고 **개별 글 상세 페이지는 없음.**
  → 개선 과제: 클릭 시 본문을 보여주는 상세 뷰(모달 또는 `post.html?id=`) 추가 검토.

## 7. 디자인 방향 (Codex가 고도화할 부분)

현재 테마: **Claude(Anthropic) 스타일**
- 배경: 따뜻한 크림/아이보리 (`--bg: #faf9f5`, `--bg-soft: #f0eee6`)
- 포인트: 코랄/클레이 (`--accent: #d97757`, `--brand: #c9684a`)
- 제목: 세리프 계열(`--font-serif`), 본문: Pretendard 산세리프
- 톤: 정갈하고 차분하며 따뜻함 (아동·교육 비영리에 어울리게)

### 고도화 아이디어(원하면 반영)
- 히어로에 일러스트/사진 또는 부드러운 그래픽 요소 추가
- 활동 사진 갤러리 섹션
- 스크롤 등장 애니메이션(가벼운 CSS/IntersectionObserver 수준)
- 로고 이미지 적용(현재는 "꿈" 글자 타일)
- 게시글 상세 보기, 접근성(a11y)·SEO 메타 보강

> ⚠️ 톤을 크게 바꾸려면 사용자와 확인. 색은 `:root` 변수만 바꿔도 전체 반영됨.
