# 교육성장네트워크 꿈들 — 홈페이지

비영리 민간단체 **교육성장네트워크 꿈들**의 새 공식 홈페이지입니다.
기존 회원형 PHP 사이트(꿈들.com)를 **가볍고 무료로 운영 가능한 정적 사이트**로
새로 만들고 콘텐츠를 이관하기 위한 저장소입니다.

## 🧩 구성

| 파일 | 내용 |
|------|------|
| `index.html` | 홈(소개·최근 소식·후원 CTA) |
| `about.html` | 단체소개(미션·연혁) |
| `projects.html` | 사업·프로젝트(하늘오름어린이풍물단 등) |
| `news.html` | 소식·보도자료·공지 목록 |
| `sponsor.html` | 후원안내 |
| `contact.html` | 문의 |
| `assets/style.css` | 공통 디자인(반응형) |
| `assets/site.js` | 공통 헤더/푸터, 모바일 메뉴 |
| `assets/posts.js` | **게시글 데이터** (이관·추가는 여기서) |

> 빌드 도구·서버가 필요 없습니다. HTML 파일을 그대로 열면 동작합니다.

## ✍️ 콘텐츠(게시글) 이관·추가 방법

`assets/posts.js`의 `POSTS` 배열에 항목을 추가하면
홈의 '최근 소식'과 소식 페이지에 자동으로 표시됩니다.

```js
{
  category: "소식",          // 소식 | 보도자료 | 프로젝트 | 공지사항
  title: "글 제목",
  date: "2026-06-30",        // YYYY-MM-DD
  body: "본문 내용...",       // 기존 사이트에서 옮겨온 본문 (선택)
  link: "https://..."        // 외부 원문 링크가 있으면 (선택)
}
```

기존 꿈들.com(그누보드) 게시판의 **소식 32 · 프로젝트 26 · 보도자료 15 = 총 73건**을
제목·작성일·본문·이미지까지 `posts.js` / `assets/img/posts/` 로 **이관 완료**했습니다.
목록의 글을 누르면 본문 상세보기(모달)가 열립니다. (자세한 내용은 `MIGRATION_NOTES.md`)

## 🚀 무료 배포 (GitHub Pages)

1. 이 브랜치를 `main`에 머지합니다.
2. GitHub 저장소 → **Settings → Pages**
3. **Source: Deploy from a branch** → Branch: `main` / 폴더 `/(root)` → Save
4. 잠시 후 `https://<사용자명>.github.io/kkumddel/` 에서 공개됩니다.
5. (선택) **꿈들.com 도메인 연결** — Settings → Pages → Custom domain 에 도메인을
   입력하고, 도메인 등록처에서 DNS(CNAME/A 레코드)를 GitHub Pages로 설정합니다.

## ⚠️ 기존 사이트 대비 변경점

정적 사이트라 **서버 기능은 외부 서비스로 대체**합니다.

| 기존 기능 | 새 사이트에서의 처리 |
|-----------|----------------------|
| 회원 로그인 / 포인트 (56명) | ❌ 미지원 — 소개형 사이트에는 불필요 |
| 방문자 글쓰기 게시판·댓글 | 📄 읽기 전용 콘텐츠로 전환 |
| 후원신청 / 실시간문의 | 🔗 구글폼·이메일·전화로 대체 |
| 소식·보도·공지·프로젝트 글 | ✅ `posts.js`로 이관 |
