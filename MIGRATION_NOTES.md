# 게시글 본문 이관 — 완료 보고

> 갱신: 2026-07-01 · 브랜치: `claude/kkumdeul-post-migration-paphm4`

기존 꿈들.com(그누보드 기반 회원형 PHP 사이트)의 **게시글 본문을 새 정적
사이트로 이관 완료**했습니다.

## ✅ 이관 결과

| 분류 | 기존 게시판 | 건수 |
|------|------------|------|
| 소식 | `3_1_1_1` | 32 |
| 프로젝트 | `2_1_1_1` | 26 |
| 보도자료 | `4_1_1_1` | 15 |
| **합계** | | **73** |

- 모든 글의 **제목·작성일·본문(HTML)** 을 `assets/posts.js`의 `POSTS` 배열로 이관.
- 본문에 포함된 **꿈들 자체 이미지 242개**를 `assets/img/posts/<게시판>_<글번호>/`
  로 내려받아 로컬화(약 74MB). 새 사이트는 외부 도메인 없이 이미지가 표시됩니다.
- 소식 페이지·홈 '최근 소식'·프로젝트 페이지에서 글을 누르면 **본문 상세보기(모달)**
  가 열립니다. (기존엔 본문 표시 기능이 없었음 → 신규 추가)

## 🔧 이관 방법 (재현용)

꿈들.com은 그누보드 사이트로, 게시글 페이지가 **JS 리다이렉트 anti-bot 게이트**를
사용하고 공유호스팅 특성상 세션/IP 검증이 있습니다. 다음 순서로 우회했습니다.

1. 새 세션으로 `http://xn--cw0by0h.com/index.php` 를 먼저 요청해 세션 쿠키 확보.
2. 같은 세션 + Referer 로 `http://xn--cw0by0h.com/bbs/board.php?bo_table=<board>&wr_id=<id>`
   요청(정상 본문이 올 때까지 재시도).
3. 본문 컨테이너 추출:
   - 소식/프로젝트 스킨: `div[style*="border:1px solid #e9e9e9"]`
   - 보도자료 스킨: `span#writeContents`
   - 제목: `<head>`의 주석 처리된 `<!-- <title>…</title> -->` (스킨 무관, 신뢰도 높음)
4. 본문 내 `/data/...` 이미지를 로컬로 다운로드 후 경로 재작성.

## ℹ️ 참고 / 남은 항목

- **프로젝트(2_1_1_1) 글 5건**은 원본 스킨에 작성일이 없어, 기수(期)·제목의
  연도로 **근사 날짜**를 부여했습니다(정확한 날짜를 주시면 교체).
- **외부 언론사 사진**(제민일보·제주의소리·헤드라인제주 등)과 wix 이미지 37개는
  원본이 외부 서버에 있어 로컬화하지 못했습니다. `http→https` 로 승급했고,
  링크가 살아 있으면 표시되며 사라진 경우 자동 숨김 처리됩니다(본문 텍스트는 보존).
- 유튜브 임베드 2건은 그대로 유지했습니다.

## 초기 조사 메모 (해결됨)

초기에 이 환경의 egress 프록시가 세션/JS 게이트를 통과하지 못해 도메인의
기본 vhost(제주철인3종협회 페이지)만 응답받아, 잠시 "도메인이 넘어갔다"고
오판했습니다. 세션·Referer·JS 게이트 우회 후 정상적으로 꿈들 콘텐츠에
접근·이관했습니다.
